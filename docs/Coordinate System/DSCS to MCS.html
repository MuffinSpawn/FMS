Coordinate System.lvlib:DSCS to MCS.vi

This VI uses the specified transform matrix to convert a list of Detector Solenoid Coordinate System (DSCS) coordinates (row vector matrix) into a list of Mapper Coordinate System (MCS) coordinates. This transform matrix should not be confused with the transform matrix that takes Laser Tracker Coordinate System (LTCS) coordinates to the DSCS. This involves converting the cylindrical DSCS coordinates to Cartesian coordinates, applying the transform matrix, and then converting the Cartesian MCS coordinates to cylindrical coordinates.
