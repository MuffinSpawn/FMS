Coordinate System.lvlib:Mapper Coordinates to Propegator.vi

This VI creates a 4x4 propegation matrix from the relative rotation and linear displacement applied to the mapper. When used with the Propagate Coordinates VI, the propegation matrix takes a list of cylindrical Detector Solenoid Coordinate System vectors (row vector matrix), converts them to Cartesian coordinates, propegates them through the same rotation and linear displacement, and then converts them back to cylindrical coordinates.

 Mapper Coordinates

Coordinates of the mapper comprising a rotation angle (theta) and a linear displacement (z).

    		theta (deg)

    		z (mm)

Flip z-axis

If True, the direction of the z axis of the mapper coordinate system is assumed to be opposite that of the direction of the detector solenoid coordinate system. Otherwise, the two z-axes point in the same direction.

Propagation Matrix

When applied to a column 4-vector matrix in the Detector Solenoid Coordinate System, (rho, theta, z), the vector will be propegated to a point that cooresponds to the rotation and linear displacement specified in Mapper Coordinates.

    		Real Matrix Element
