 DSCS Coordinate List in

Input Detector Solenoid Coordinate System row vectors (rho, theta, z).

    		phi (rad)

Propagation Matrix

When applied to a Cartesian column 4-vector matrix in the Detector Solenoid Coordinate System, (rho, theta, z), the vector will be propegated to a point that cooresponds to the rotation and linear displacement specified in Mapper Coordinates. Conversion to/from Cartesian coordinates is done automatically.

    		Real Matrix Element
